# Introduction to JavaScript Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a reusable block of code designed to perform a particular task.

## Basic Function Syntax

Let's start with a simple function declaration:

<CodeBlock language="javascript" title="basic-function.js">
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("Alice")); // Output: Hello, Alice!
</CodeBlock>

<Note type="info" title="Function Components">
A function has three main parts:
- **Name**: The identifier used to call the function
- **Parameters**: Input values the function accepts
- **Body**: The code that executes when the function is called
</Note>

## Arrow Functions

ES6 introduced arrow functions, which provide a more concise syntax:

<CodeBlock language="javascript" showLineNumbers>
// Traditional function
function add(a, b) {
  return a + b;
}

// Arrow function
const add = (a, b) => a + b;

// With single parameter, parentheses are optional
const square = x => x * x;
</CodeBlock>

<AIExplainer concept="arrow functions">
Arrow functions not only provide shorter syntax but also have different behavior with the `this` keyword compared to regular functions.
</AIExplainer>
